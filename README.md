# SAVE GAME - Sistema de Gestão de Jogos

**Desenvolvido por:** Fernanda Costa Moraes

Este é um sistema CRUD completo para gerenciamento de jogos, desenvolvido como requisito para o Trabalho 5.
## Como rodar o projeto
### 1. Banco de Dados
- Importe o arquivo `banco_dados.sql` no seu MySQL Workbench. 

### 2. Servidor (Backend)
- Acesse a pasta `backend` pelo terminal.
- Execute `npm install` para instalar as dependências.
- Execute `node index.js` para iniciar o servidor na porta 3001.

### 3. Interface (Frontend)
- Acesse a pasta `frontend` pelo terminal.
- Execute `npm install` para instalar as dependências.
- Execute `npm run dev` para iniciar a aplicação.
